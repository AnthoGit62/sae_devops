Outils Utilisés



  • Firebase Hosting

    Firebase Hosting est un service de Google qui permet d'héberger facilement des sites web statiques et dynamiques avec un CDN performant. Il prend en charge HTTPS, le déploiement rapide via la ligne de commande et l'intégration avec d'autres services Firebase. Idéal pour les applications web modernes, il facilite la mise en ligne et la gestion des versions.



Architecture du projet



  • Navbar

    Cet navbar à était créer pour comme son nom l'indique naviguer sur le sites. Il est présent sur toutes les pages du site.

  • Header

    Le header permet d'avoir un aperçu de la page où l'on se trouve. Il est présent sur toutes les pages du site.

  • Corps

    Dans le corps de la page HTML on y retrouve l'essentiel de l'information. Il est présent sur toutes les pages du site.

  • Footer

    Enfin, le footer est le pied de page qui regroupe les cookie, politique de Confidentialité, conditions d'utilisations et le copyright. Il est présent sur toutes les pages du site.
